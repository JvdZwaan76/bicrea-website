# Favicons

Copy the favicon set from the main bicrea-website repo's `/favicon/` directory.

## Required files

- `favicon.svg`
- `favicon.ico`
- `favicon-96x96.png`
- `apple-touch-icon.png`
- `web-app-manifest-192x192.png`
- `web-app-manifest-512x512.png`
- `site.webmanifest`

## Quick copy

```bash
cp ../bicrea-website/favicon/* ./favicon/
```
